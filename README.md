# e-commerce-application

### [Demo](https://e-commerce-app-davidk55.netlify.app/)

## Features

- JWT Authentication
  - In memory Access Token
  - Refresh Token as HTTP only Cookie
- Role based Authorization
- E-mail and Password Validation

## Technologies

### Backend

- Spring Boot
- Spring Security
- Lombok
- JJWT
- PostgreSQL

### Frontend

- Figma
- Webpack
- Tailwind CSS
- Headless UI
- Typescript
- Axios
- React
- React Router v6
- Postman
